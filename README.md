## Cómo reproducir el proyecto

1. Instalar [VirtualBox](https://www.virtualbox.org/) en Windows 11
2. Descargar la ISO de [Windows Server 2022](https://www.microsoft.com/es-es/evalcenter/evaluate-windows-server-2022)
3. Crear una VM con 4 GB RAM, 2 CPU y 50 GB de disco
4. Seguir los pasos detallados en [Avance2.md](./Avance2.md)
